# ecommerce-website-php-native

A fully functional and secure e-commerce web application built using native PHP and MySQL.
This project includes a complete user-facing store and a powerful admin panel, focusing on security, clean structure, and real-world functionality.

What This Project Does

This application simulates a real online store where:

Users can browse products, create accounts, place orders, and track their purchases

Admins can manage products, categories, users, and orders from a dedicated dashboard

All sensitive data (like passwords) is handled securely using best practices
